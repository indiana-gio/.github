# Welcome to the Indiana Geographic Information Office's public code organization.

The Indiana Geographic Information Office (IGIO) is the clearinghouse for GIS in the state of Indiana. This GitHub organization is meant to demonstrate code and methods for working with the IGIO's publicly available GIS data. The IGIO provides no support or warranty for the code and intends it for educational purposes only.
